## Ипользованные инструменты:

- React
- TypeScript
- Redux-saga
- axios
- Vite
- Material-UI

### Установка и запуск

Установить Node.js  
Node Version Manager

```bash
nvm install
```

Fast Node Manager

```bash
fnm install
fnm use
```

Установить зависимости:

```bash
npm install
```

Сбилдить проложение:

```bash
npm run build
```
